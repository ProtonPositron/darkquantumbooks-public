# Dark Quantum Books / Квант тьмы

**Dark Quantum Books** / **«Квант тьмы»** is a publishing project for atmospheric speculative fiction: stories after which the familiar world feels less flat.

**«Квант тьмы»** — издательский проект атмосферной фантастики под другим углом: историй, после которых привычный мир кажется не таким плоским.
